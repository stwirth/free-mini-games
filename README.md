# Free Mini Games

A collection of free, browser-based mini games playable directly in your browser — no installation required.

The site is statically served via [GitHub Pages](https://stwirth.github.io/free-mini-games/).

> **This repository is 100% AI-driven.** All code and content is generated exclusively by AI agents. No manual code edits are allowed.

## Instructions for coding agents

* all commits should include the full (verbatim) prompt that lead to the commit as commit messages
* every html site should contain a side-bar with a changelog, containing commit SHAs and commit messages, linking to the respective commit
