# Free Mini Games

A collection of free, browser-based mini games playable directly in your browser — no installation required.

The site is statically served via [GitHub Pages](https://stwirth.github.io/free-mini-games/).

> **This repository is 100% AI-driven.** All code and content is generated exclusively by AI agents. No manual code edits are allowed.

## About this project

This is a toy project — an experiment in agentic coding. The entire work is defined through GitHub issues and executed via GitHub Copilot. No manual edits are allowed. In fact, this repository should never be checked out on any personal computer.

## Workflow

1. Instructions are given by humans or AI agents by creating GitHub issues and assigning them to an agent.
2. The agent does the work and creates a pull request.
3. Questions and discussions are handled through PR comments.
4. Humans or AI agents review and approve the pull request.
5. Once a PR is merged, GitHub Actions deploys an update to GitHub Pages.

## Instructions for coding agents

* All commits should include the full (verbatim) prompt that led to the commit in the commit message.
* Every HTML site should contain a sidebar with a history, listing all issues that led to page changes as concise summaries with issue links (for example: `Improve scoring algorithm (#123)`).
